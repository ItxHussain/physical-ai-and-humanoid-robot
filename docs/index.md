# Physical AI Modules Book

Welcome to the comprehensive guide on Physical AI modules for humanoid robots. This book explores the four core modules that enable humanoid robots to achieve perception → planning → navigation → manipulation:

## Core Modules

1. [**ROS 2 (Robotic Nervous System)**](./physical-ai-modules/module-1-ros2.md) - Middleware for robot control
2. [**Digital Twin (Gazebo & Unity)**](./physical-ai-modules/module-2-digital-twin.md) - Physics and sensor simulation
3. [**NVIDIA Isaac (AI-Robot Brain)**](./physical-ai-modules/module-3-nvidia-isaac.md) - AI-powered robot intelligence
4. [**Vision-Language-Action (VLA)**](./physical-ai-modules/module-4-vla.md) - Cognitive planning and reasoning
